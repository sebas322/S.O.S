# S.O.S
Descripción: 
Es una aplicación Android llamada S.O.S, cuyo propósito es emitir señales de auxilio utilizando el flash de la cámara del dispositivo y sonidos sincronizados en código Morse. Esta herramienta está diseñada para ser utilizada en situaciones de emergencia donde el usuario requiera asistencia inmediata.

Requisitos:

minSdk = 21
La app puede instalarse en dispositivos con Android 5.0 Lollipop o superior.

targetSdk = 35
La app está optimizada para correr en Android 14 (API 35).

Compatibilidad mínima: cualquier dispositivo con Android 5.0 (Lollipop) o más reciente podrá instalar y ejecutar la aplicación sin ningun inconveniente.
